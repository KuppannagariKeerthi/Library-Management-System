# Library Management System (Java)

## Project Description

The Library Management System is a simple console-based application developed using Java.
This project helps to manage basic library operations such as adding books, viewing books, registering students, issuing books, and returning books.

The system maintains details of books and students and allows a librarian to perform essential library functions easily through a menu-driven interface.


## Features

• Add new books to the library
• View all available books
• Search books using Book ID
• Delete books from the library
• Register new students
• View registered student details
• Issue books to students
• Return books to the library


## Technologies Used

• Java
• Object Oriented Programming (OOP)
• Arrays and Basic Data Structures

## How to Run the Project

1. Download or clone the repository.

2. Compile the program

javac LibraryManagementSystem.java

3. Run the program


java LibraryManagementSystem

4. Follow the menu options displayed in the console.
